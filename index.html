Installation Instructions

Windows Installation

  1. Download the Windows ZIP file above
  2. Extract the contents to a folder of your choice (e.g., C:\Games\TIB)
  3. Run TIB-Remastered.exe
  4. Create an account or log in with existing credentials
  5. Select the "Private Server" option
  6. Start playing!

macOS Installation

  1. Download the macOS ZIP file above
  2. Extract the contents (double-click the ZIP file)
  3. Move TIB-Remastered.app to your Applications folder
  4. Right-click the app and select "Open" (first time only, to bypass Gatekeeper)
  5. Create an account or log in with existing credentials
  6. Select the "Private Server" option
  7. Start playing!
